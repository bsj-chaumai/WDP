## A. 60秒で全体像

```text
Visの今
  └─ 売上の多くはワークプレイスデザイン（移転・改装）
  └─ 第2の柱として エクスペリエンスデザイン（XD）× WDP を育てたい

WDPの今
  └─ 営業支援は強い（受注率↑ → 間接売上効果）
  └─ 一方でショット型（現状把握）のまま → 有料はほぼなし

直近のゴール（FY2026 = 検証／MVP）
  └─ XDスターター ＋ WDPのAfterスポット利用
  └─ まだ本格的なSaaS化の年ではない
```

**一言：**  
WDPは **受注（勝ち案件）には効いている**。  
まだ足りないのは **効果の証明 / 有料化 / XDとの接続**。

## B. 用語（資料の言い方のまま）

| 資料の用語 | かんたんに言うと |
|------------|------------------|
| **ワークデザイン** | **ワークプレイスデザイン × エクスペリエンスデザイン** |
| **ワークプレイスデザイン（WD）** | オフィス空間をつくる仕事（請負・労働集約型） |
| **エクスペリエンスデザイン（XD）** | 移転後の **運用・意識・活動** を支援し、効果の形骸化を防ぐ |
| **WDP** | WORK DESIGN PLATFORM（人の気持ち / 人の動き / 空間） |
| **WDS** | ワークデザインサーベイ（WDP内のサーベイ） |
| **ショット型** | 1回きりの利用 |
| **Before / After** | 前後比較による **効果検証** |
| **間接売上効果** | 受注率が上がった分だけを数えた売上効果 |
| **営業支援ツール / 営業・提案基盤** | いま証明できているWDPの役割 |
| **有料契約 / 有料化** | 顧客がWDP利用にお金を払うこと |
| **検証（MVP）** | 最小構成で価値を確かめる年 |
| **PMF** | 繰り返し売れる状態 |
| **自走** | 顧客がコンサル依存を減らし、自分たちで回すこと |
| **SaaS型** | 継続利用の課金型プラットフォーム |

## C. 本文（この順で読む）

### C1. 現状 — いまどこにいるか

#### ① Vis（事業）

| | 内容 |
|--|------|
| 長期目標 | **2030年 売上高250億** |
| いまの売上（FY2025） | 多くは **オフィスの移転・改装** |
| ビジネスモデル | **請負契約** ＋ **労働集約型**。市場は良く、売上は右肩上がり |
| 弱点 | 人数を増やし続ける必要がある。デザインでの差別化は競合に近づかれている |
| 位置づけの変化 | 2023年：デザイナーズオフィス → **ワークデザイン** |
| 定義 | 2025年半期総会：**ワークデザイン＝WD × XD** |
| XD | **2026/4** 正式ローンチ（資料上は予定） |

#### ② WDP（ツール）

| | 内容 |
|--|------|
| 主に使っている機能 | **サーベイ** |
| 利用量 | およそ **年間100件ペース** |
| 有料 | **まだ２件程度**（マネタイズには程遠い） |
| 使い方 | **現状把握** → **ショット型**（継続利用になっていない） |
| すでに出ている価値 | **営業支援**：受注率アップ → **間接売上効果** |

**受注率（2025/10時点・今期完工ベース）**

| 案件 | 通常 | WDP活用時 | 差分 |
|------|------|-----------|------|
| コンペ | 30.9% | **53.8%** | **＋22.9pt** |
| 特命・ナーチャリング | 49.6% | **75.0%** | **＋25.4pt** |

**間接売上効果**  
`WDP活用完工売上 × 受注率上昇分` ＝ **約146,612,083円**（約1.46億。本文では約1.4億とも記載）

#### ③ Vis社内の活用

| | 数字 |
|--|------|
| オフィスリニューアル | およそ **年間500件** |
| WDS | **月平均10件** |
| ワークショップ | **年間5〜6件** |
| 実態 | 意識の高い人に偏る。未利用の人も多い |

### C2. 課題 — なぜこのままだと足りないか

3層で見る：

```text
1) WDPの直接収益が弱い
     有料〜2件 / ショット型 / Before→Afterが足りない

2) 営業の現場に広がっていない
     500件に対し WDS 10件/月・WS 5〜6件/年
     提案リードタイムは3〜4週間 → WDSを間に合わせにくい

3) 戦略上のプレッシャー
     WDの差別化が薄れる → XDが必要
     だが開発サイクルが重い → WDPがXDに追いつきにくい
```

| 課題 | 中身 | 影響 |
|------|------|------|
| **ショット型の現状把握** | 提案前の現状把握が中心 | 継続利用にならない |
| **マネタイズが弱い** | 有料まだ２件程度 | 直接売上がほぼ出ない |
| **Before→After不足** | マネタイズには before and after 拡大とXD連携が必要 | 効果をツールで示しきれない |
| **社内活用の偏り** | 500件に対し10件/月・5〜6件/年 | スケールしない |
| **提案リードタイム** | 3〜4週間ではサーベイ反映が難しい | 受注支援の機会を逃す |
| **開発が重い** | XP/XD向けの軽い追加がしづらい | XDが先に進み、WDPが遅れる |

---

### C3. 目指す姿 — どこへ向かうか

#### 大きな方向

```text
残す：WDP = 営業支援（受注の一助）
足す：WDP = Before→After の効果検証（XDとつなぐ）
目指す：ショット型 → 継続利用 → 有料化 → SaaS型（顧客自走のタイミング）
```

#### XD MVP（FY2025–2026）の中身

- サーベイの **before and after効果検証**
- ワークショップ（アブストラクションラダー）
- リニューアル後イベントを最低 **2** 企画
- サーベイ項目の目標設定とスコアゴール
- フォロー ＋ 最終レポート  
→ 入口価格の目安 **約100万円**。通常のリニューアル提案にセットし、**2026/4** から開始予定

#### 年度別の売上計画

| 年度 | XD | WDP | 合計 | フェーズ | その年にやること |
|------|----|-----|------|----------|------------------|
| **FY2026** | 0.55億 | 0.2億 | **0.7億** | **検証（MVP）** | スターター **40件**。年間契約はPoC最大1社。WDPは **After中心のスポット利用** |
| FY2027 | 1.5億 | 0.5億 | 2.0億 | **PMF** | スターター→フルXD→年間。WDPは **自走** の価値が出始める |
| FY2028 | 3億 | 1億 | 4億 | スケール開始 | XDは年間契約が主力。WDPは **SaaS商品化**。開発は **完全内製化** |
| FY2029 | 6.0億 | 2億 | 8.0億 | 拡張 | ストック比率が上がる |
| FY2030 | 15.45億 | 4億 | **19.45億** | 事業確立 | 250億構想の第2柱（目安として連結 **約25億** も記載） |

#### FY2026で覚えること（3点）

1. **XD**：誰に・何が・なぜ刺さるかを検証（スターター40件）  
2. **WDP**：**After中心のスポット利用** ＋ 数値化文化  
3. **有料化**：**beforeからafterの訴求** につなげる（まだSaaSフルではない）

### C4. よくある誤解

| 誤解しやすい理解 | 資料上の正しい意味 |
|------------------|-------------------|
| 有料2件＝オフィス案件が2件 | **WDPの有料契約が約2件** |
| 1.46億＝WDPの販売売上 | **間接売上効果**（受注率上昇分） |
| FY2026でSaaS完成 | FY2026は **MVP / Afterスポット**。SaaSは主に **FY2028** |
| WDPは提案前支援をやめる | **受注支援は残す**。After / XD / 有料を **足す** |

## E. 1枚メモ（共有用）

```text
【現状】WDが主力（約500件）／WDPはサーベイのショット型（約100件・有料約2）
        ただし受注率↑ → 間接約1.46億（営業支援は証明済み）

【課題】Before→After不足 → 有料・XD接続が弱い
        社内活用が偏る／提案3〜4週間ではサーベイが間に合いにくい
        開発サイクルが重い

【方針】ワークデザイン＝WD × XD
        WDP：受注の一助は維持 ＋ After効果検証を追加
        → 継続利用・有料化・SaaSへ

【年表】FY2026 MVP 0.7億（Afterスポット／スターター40）
        FY2027 PMF → FY2028 SaaS・内製化 → FY2030 第2の柱
```
