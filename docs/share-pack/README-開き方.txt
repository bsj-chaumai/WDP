【このZIPの開き方】

1. ZIPを解凍（展開）してください
   ※ ZIPの中から直接HTMLを開くと、リンクが白画面になることがあります

2. 解凍後、同じフォルダに次の2ファイルがあることを確認
   - 06-wdp-wd-xd-flow-ja.html（図解：メイン）
   - 05-brief-team-vis-genjo-to-mokuhyo-ja.html（現状サマリー）

3. 06-wdp-wd-xd-flow-ja.html をブラウザで開く
   （Chrome / Edge / Safari 推奨）

4. リンクの種類
   a) 「現状サマリー（JA）」
      → 同じフォルダのHTMLを開きます（ネット不要）
   b) WDP公式 / XD公式
      → インターネットが必要です
      → 白画面のときは、ブラウザ新規タブにURLを貼って開いてください
         https://vis-produce.com/service/data-solution/wdp
         https://vis-produce.com/lp/experience-design

【よくある原因】
- 図解HTMLだけ送ってサマリーHTMLが無い
- ファイル名を変更した
- 別フォルダに分けて置いた
- ZIPを解凍せずに開いた
- 社内ネット制限でVis公式サイトにアクセスできない
