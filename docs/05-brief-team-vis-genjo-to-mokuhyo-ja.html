A. 60秒で全体像

Visの今
  └─ 売上の多くはワークプレイスデザイン（移転・改装）
  └─ 第2の柱として エクスペリエンスデザイン（XD）× WDP を育てたい

WDPの今
  └─ 営業支援は強い（受注率↑ → 間接売上効果）
  └─ 一方でショット型（現状把握）のまま → 有料はほぼなし

直近のゴール（FY2026 = 検証／MVP）
  └─ XDスターター ＋ WDPのAfterスポット利用
  └─ まだ本格的なSaaS化の年ではない

一言：WDPは受注（勝ち案件）には効いている。まだ足りないのは効果の証明 / 有料化 / XDとの接続。

B. 用語（資料の言い方のまま）

資料の用語かんたんに言うと
ワークデザインワークプレイスデザイン × エクスペリエンスデザイン
ワークプレイスデザイン（WD）オフィス空間をつくる仕事（請負・労働集約型）
エクスペリエンスデザイン（XD）移転後の運用・意識・活動を支援し、効果の形骸化を防ぐ
WDPWORK DESIGN PLATFORM（人の気持ち / 人の動き / 空間）
WDSワークデザインサーベイ（WDP内のサーベイ）
ショット型1回きりの利用
Before / After前後比較による効果検証
間接売上効果受注率が上がった分だけを数えた売上効果
営業支援ツール / 営業・提案基盤いま証明できているWDPの役割
有料契約 / 有料化顧客がWDP利用にお金を払うこと
検証（MVP）最小構成で価値を確かめる年
PMF繰り返し売れる状態
自走顧客がコンサル依存を減らし、自分たちで回すこと
SaaS型継続利用の課金型プラットフォーム

C2. 課題 — なぜこのままだと足りないか

1) WDPの直接収益が弱い
     有料〜2件 / ショット型 / Before→Afterが足りない

2) 営業の現場に広がっていない
     500件に対し WDS 10件/月・WS 5〜6件/年
     提案リードタイムは3〜4週間 → WDSを間に合わせにくい

3) 戦略上のプレッシャー
     WDの差別化が薄れる → XDが必要
     だが開発サイクルが重い → WDPがXDに追いつきにくい
課題中身影響
ショット型の現状把握提案前の現状把握が中心継続利用にならない
マネタイズが弱い有料まだ２件程度直接売上がほぼ出ない
Before→After不足マネタイズには before and after 拡大とXD連携が必要効果をツールで示しきれない
社内活用の偏り500件に対し10件/月・5〜6件/年スケールしない
提案リードタイム3〜4週間ではサーベイ反映が難しい受注支援の機会を逃す
開発が重いXP/XD向けの軽い追加がしづらいXDが先に進み、WDPが遅れる

C3. 目指す姿 — どこへ向かうか

大きな方向

残す：WDP = 営業支援（受注の一助）
足す：WDP = Before→After の効果検証（XDとつなぐ）
目指す：ショット型 → 継続利用 → 有料化 → SaaS型（顧客自走のタイミング）

XD MVP（FY2025–2026）の中身

  • サーベイのbefore and after効果検証
  • ワークショップ（アブストラクションラダー）
  • リニューアル後イベントを最低2企画
  • サーベイ項目の目標設定とスコアゴール
  • フォロー ＋ 最終レポート

→ 入口価格の目安 約100万円。通常のリニューアル提案にセットし、2026/4 から開始予定

年度別の売上計画

年度XDWDP合計フェーズその年にやること
FY2026 0.55億 0.2億 0.7億 検証（MVP） スターター40件。年間契約はPoC最大1社。WDPはAfter中心のスポット利用
FY2027 1.5億 0.5億 2.0億 PMF スターター→フルXD→年間。WDPは自走の価値が出始める

FY2026で覚えること（3点）

  1. XD：誰に・何が・なぜ刺さるかを検証（スターター40件）
  2. WDP：After中心のスポット利用 ＋ 数値化文化
  3. 有料化：beforeからafterの訴求につなげる（まだSaaSフルではない）

E. 1枚メモ（共有用）

【現状】WDが主力（約500件）／WDPはサーベイのショット型（約100件・有料約2）
        ただし受注率↑ → 間接約1.46億（営業支援は証明済み）

【課題】Before→After不足 → 有料・XD接続が弱い
        社内活用が偏る／提案3〜4週間ではサーベイが間に合いにくい
        開発サイクルが重い

【方針】ワークデザイン＝WD × XD
        WDP：受注の一助は維持 ＋ After効果検証を追加
        → 継続利用・有料化・SaaSへ

【年表】FY2026 MVP 0.7億（Afterスポット／スターター40）
        FY2027 PMF → FY2028 SaaS・内製化 → FY2030 第2の柱
