1. ヴィスのサービスと現状

ヴィスの価値提供は「デザイナーズオフィス」から ワークデザインへ切り替わっています。 定義は ワークデザイン ＝ ワークプレイスデザイン（WD）× エクスペリエンスデザイン（XD）。 事業計画

WD：ワークプレイスデザイン（いまの主力）

  • 公式：プログラミング → デザイン → クリエイト → アップデート 公式XD
  • 現状：FY2025売上の多くは移転・改装プロジェクト 事業計画
  • モデル：請負契約＋労働集約型。市場は良く売上は右肩上がり 事業計画
  • 規模感：オフィスリニューアルは年間約500件 事業計画

XD：エクスペリエンスデザイン（成長の柱）

  • 公式：ビジョン策定 → 計画 → 運用 公式XD
  • 役割：空間だけでなく体験をデザインし、人と企業の関係を持続的に育む 公式XD
  • 現状：資料上、2026年4月に正式サービスローンチ予定 事業計画
  • 狙い：リニューアル後の形骸化を防ぎ、オフィス投資の効果を最大化 公式XD / 事業計画

WDP：WORK DESIGN PLATFORM（ツール）

  • 公式：人の気持ち / 人の動き / 空間を可視化。サーベイ・空間稼働率分析・コストシミュレーション 公式WDP
  • 公式：移転・改装の前後で価値を数値化し、人的資本投資の効果検証ができる 公式WDP
  • 実態（資料）：主にサーベイ利用。年間約100件ペース。有料契約はまだ約2件 事業計画

一言で現状：売上はWD中心。第2の柱としてXD + WDPを育てる必要がある、というのが資料の立場です。 事業計画

2. WDPの役割・現状・受注率・課題（サービス別）

受注率の数字は、資料上ワークプレイスデザイン事業（WD案件）での実績です。 XD単体の受注率は元資料に記載がありません。

WDにおけるWDP

観点 内容
役割 提案初期・ナーチャリングで使い、仮説ではなくデータに基づく提案にする。 営業・提案基盤／受注率向上による間接売上効果。 事業計画
現状 主用途は現状把握。ショット型（1回）で継続利用につながりにくい。 社内WDSは月平均約10件（徐々に拡大）。 事業計画
受注率 2025/10時点・今期完工ベース：
コンペ：通常 30.9% → WDP活用時 53.8%（＋22.9pt）
特命・ナーチャリング：通常 49.6% → WDP活用時 75.0%（＋25.4pt） 事業計画
間接効果 算出：WDP活用完工売上 × 受注率上昇分。 合計約 146,612,083円（約1.46億。本文では約1.4億とも記載）。 事業計画
課題 ・提案リードタイム約3〜4週の中でWDS結果を反映しにくい
・ショット型のままでは有料化しにくい 事業計画

XDにおけるWDP

観点 内容
役割 公式：ビジョン策定でWDPを活用（気持ち／動き／空間）。 プロジェクト前後で測定・分析し効果検証する。 公式XD
事業計画：XDサービスをWDPを活用しながら提供。マネタイズには before and afterの拡大とXD掛け合わせが必要。 事業計画
現状 XDは資料時点では正式ローンチ前（2026/4予定）。 MVPの中核はサーベイの before and after 効果検証。 事業計画
受注率 記載なし（元資料にXD単体の受注率実績は無い）。
課題 ・市場認知がまだ少ない中で提案しても採用されにくい想定
・だからMVPは約100万円程度の低ハードルで、2026/4提案から社内ルール化予定
・事前にプロトタイプ提案を約3件 事業計画

3. ヴィスの課題と、改善したいこと

事業構造の課題（WD中心のままでは伸びにくい）

課題資料の内容
労働集約 コンサルタント／クリエイターの数を増やし続ける必要がある 事業計画
差別化の縮小 高いデザイン性での差は類似企業に埋まりつつある 事業計画
空間設計に留まる 出社率・コラボ等の課題解決を目指しても、単なる空間設計コンサルに留まりやすい 事業計画
効果の形骸化 公式XDも、満足度低下・施策継続困難・席の固定化／形骸化を課題として提示 公式XD

WDP / 活用の課題

課題資料の内容
マネタイズ遠い 有料契約まだ約2件。ショット型の現状把握中心で継続利用にならない 事業計画
Before→After不足 マネタイズには before and after 件数拡大とXD連携が必要 事業計画
社内浸透の偏り WDS月約10件、WS年5〜6件。一方Renew約500件。意識の高い人に偏る 事業計画
提案タイミング 3〜4週の提案期間内にWDS結果を反映しにくい 事業計画

改善したいこと（資料が示す方向）

  • 構造改革：通常のリニューアル提案に WD部分 + XD部分を必ず盛り込む社内ルール化（2026/4〜） 事業計画
  • XD MVP：before/after効果検証 + ラダーWS + イベント最低2 + 目標スコア設定 + 最終レポート 事業計画
  • WDP：受注の一助は維持しつつ、After中心の利用・有料化・将来のSaaSへ 事業計画
  • タイミング改善：ナーチャリング／早期実施、URL形式などハードル下げ 事業計画

4. 目指す目標（年次）

長期：Vis全体 2030年 売上高250億。 その中で第2の柱として XD+WDP 連結の目安は約25億（詳細計画の2030合計は19.45億）。 事業計画

年度 XD WDP 合計 フェーズ その年に起きること
FY2026 0.55億 0.2億 0.7億 検証（MVP） スターター40件で検証。年間契約はPoC最大1社。WDPはAfter中心のスポット利用
FY2027 1.5億 0.5億 2.0億 PMF スターター→フルXD→年間契約の導線。WDPは自走ツールとして年間利用が出始める
FY2028 3億 1億 4億 スケール開始 XDは年間契約主力。WDPはSaaS商品化。開発完全内製化
FY2029 6.0億 2億 8.0億 拡張 XDの属人性低減。WDP売上の過半がストック型へ
FY2030 15.45億 4億 19.45億 事業確立 XD＝高付加価値戦略コンサル。WDPは数百社規模。第2の柱へ
